Title: Core login flow sample

The login screen should allow access only when both credentials are correct.

Required values:
- valid email: test@demo.com
- valid password: Password123
- Success is indicated by the message: "Login successful. Welcome back."
- Failure is indicated by the message: "Invalid email or password."

Expected behavior:
- Correct email + correct password should succeed.
- Correct email + wrong password should fail.
